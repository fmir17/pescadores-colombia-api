Métodos para Usuarios:

Registrar nuevo usuario

Explicación: Servicio que permite guardar un nuevo usuario en la base de datos.
Método: POST
URL: /user
Parámetro de datos: Se espera ingrese un objeto JSON, en donde se espera información de un usuario, de la siguiente forma:
user:
{
"id": "String",
"name": "String",
"familyName": "String",
"email": "String",
"city": "String"
}
Respuesta: Se espera que sea ingresado satisfactoriamente a la base de datos.

Consultar usuario existente:

Explicación: Consulta usuario con el campo id, si ese usuario está en la base de datos muestra sus datos.
Método: GET
URL: /user/id
Parámetros URL: id=[String]
Respuesta: Se espera que el sistema muestre la información del usuario consultado
Ejemplo: http://pescadores-colombia-api.herokuapp.com/user/5005]

Listar usuarios:

Explicación: Consulta todos los usuarios guardados en la base de datos
Método: GET
URL: /user/
Respuesta: Se espera que el sistema muestre la información de todos los usuarios registrados en la base de datos:

Eliminar Usuarios

Explicación: Elimina usuario con el campo id.
Método: DELETE
URL: /user/id
Parámetros URL: id=[String]
Respuesta: Se espera que el sistema elimine

Métodos para Imagenes:

Registrar nueva URL:

Explicación: Servicio que permite almacenar url's de imágenes
Método: POST
URL: /image
Parámetro de datos: Se espera ingrese un objeto JSON, el esquema es el siguiente:
image:
{
"url": "String"
}
Respuesta: se espera que sea ingresado satisfactoriamente a la base de datos.

Listar Imágenes:

Explicación: Consulta las ultimas 4 url's ingresadas
Método: GET
URL: /image
Respuesta: se espera que el sistema muestre las ultimas 4 url's ingresadas
